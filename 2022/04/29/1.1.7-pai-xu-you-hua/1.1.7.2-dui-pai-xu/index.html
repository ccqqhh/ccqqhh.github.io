由于堆排序需要使用到树这种数据结构，所以简单过一下。

7.2.1 树

树（Tree）是 n 个结点的集合，n = 0 时称为空树；若不是空树，则树 T 由被称作根 R 的结点以及 0 个或多个互不相交的非空子树 T1、T2、...、Tk 组成。

每棵子树的根被称作根 R 的子结点，而 R 是每棵子树的根的父结点。

由于 T1、T2...它们也是树，同样也是由根结点与若干个子树组成，举例：

A 的子结点有 D、E、F，G 的父结点有 C；那些没有子结点的结点，如 D、E、F、B...称为树叶结点，也称叶子结点；有子结点的结点称为分支结点，除根结点 R 外，分支节点也被称为内部结点。

树的层次：从根 R 为第一层开始算，R 的子结点 A、B、C 为第二层，A、B、C 的子结点 D、E、F、G 为第三层；最大层次称为树的高度。

7.2.2 二叉树

二叉树是一颗树，树中的每个结点最多只能有两个子结点，称为左结点、右结点。（每个结点的子节点有 0 或 1 或 2 个）

满二叉树的所有树叶节点都在最后一层，所有的分支节点都有两个子节点，此时结点总个数 = 2k - 1。（k 为树的高度）如图：

完全二叉树

二叉树性质

  1. 在第 i 层最多有 2 个结点。（1、2、4、8、16...）

  2. 高度为 k 的二叉树最多有 2k - 1 个结点。（1、3、7、15、31...）

  3. 高度为 k 的完全二叉树至少有 个结点。（1、2、4、8...）

    证明：最少结点的完全二叉树就是 (k - 1) 高度的满二叉树的结点个数 + 1 = 。

  4. 树叶结点个数为 n0，有两个子树的结点个数为 n2，有 n0 = n2 + 1。

    证明：设一共 n 个结点，每个结点都通过一条线连接着它的父结点，除了根结点，则一共有 n - 1 条线。

    定义 n1 为有一个子树的结点个数，有 n - 1 = n1 + 2 * n2；（一个结点有两个子树代表两条线，有一个子树代表一条线，把它们都加起来）

    因为二叉树结点只包括这三种类型，所以有 n = n0 + n1 + n2，

    n - 1 = n1 + 2 * n2 = n0 + n1 + n2 - 1，整理得到 n2 = n0 - 1。

  5. 具有 n 个结点的完全二叉树的高度为 ⌊log2n⌋ + 1。

    证明：设完全二叉树的高度为 k，n 一定小于 (k + 1) 高度的完全二叉树的结点个数；根据性质 3 得：，两边取对数：，整理得：，又因 k 是整数，当 log2n 为整数时，k = log2n + 1；不为整数时，k = ⌊log2n⌋ + 1。

    ⌊⌋ 是向下取整（floor），⌈⌉ 是向上取整（ceiling），如 ⌊3.5⌋ = 3。
